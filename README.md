# Budjetti


Työpöytä sovellus Windowsille. Käyttäjä pystyy pitämään kirjaa rahatilanteestaan. Ohjelmaan kirjataan kaikki kuukauden tulot ja menot ja ohjelmaa laskee paljonko rahaa on käytettävissä kuukauden aikana. Myös säästöistä voidaan pitää kirjaa.
